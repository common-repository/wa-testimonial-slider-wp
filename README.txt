Plugin to show a Bootstrap Testimonial slider for your website

== Description ==

Plugin to show a Bootstrap Testimonial slider for your website.
You can add new testimonial, edit one, or delete one.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `watestimonialsliderwp.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place shortcode [watestimonialsliderwp] in your page
